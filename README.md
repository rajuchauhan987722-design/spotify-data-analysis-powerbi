# 🎧 Spotify Top Songs Power BI Dashboard

An interactive Power BI dashboard analyzing Spotify Top Songs dataset to uncover insights into artist performance, song popularity, album trends, and music listening patterns.

---

## 📊 Project Overview

This project focuses on analyzing Spotify music data to extract meaningful business insights using Power BI.

The dashboard provides a comprehensive view of:
- Song popularity trends
- Artist performance
- Album vs Single analysis
- Explicit vs Non-explicit content
- Monthly and yearly trends

---

## 📈 Key Metrics

- 🎵 Total Songs: 789  
- 🎤 Total Artists: 342  
- ⭐ Average Popularity: 89.6  
- ⏱️ Average Duration: 3.28 minutes  

---

## 🔍 Key Insights

- Top artists dominate the platform with high song counts  
- Popular songs contribute significantly to total engagement  
- Monthly trends show fluctuations in song popularity  
- Albums and singles perform differently in terms of reach  
- Explicit content has a notable share in total songs
